# Orava

![Logo Orava](https://upload.wikimedia.org/wikipedia/commons/5/5d/Placeholder_logo_image.png)

Une application mobile intelligente permettant de générer et suivre des menus alimentaires adaptés aux besoins caloriques et nutritionnels de l'utilisateur.

> **Orava**, c’est un pont entre tradition et innovation. Inspiré par les régions naturelles d’Europe centrale et les rythmes du corps, Orava est votre allié pour retrouver un équilibre nutritionnel sur mesure, en toute intelligence.

## ✨ Objectif
Créer un outil capable de :
- Calculer les besoins nutritionnels quotidiens d'un individu
- Générer automatiquement des menus équilibrés en fonction de ces besoins
- Respecter des contraintes définies (type d'aliments, quantités, fréquence...)
- Permettre à l'utilisateur de valider ou refuser les menus proposés
- Améliorer la pertinence des menus grâce à un apprentissage machine supervisé

## 🔄 Fonctionnement du projet
1. **Base de données nutritionnelle**
2. **Générateur de menus**
3. **Système de validation par IA**
4. **Application mobile**

## 💡 Stack technique
- **Backend** : Python (Pandas, scikit-learn, Flask/FastAPI)
- **Machine Learning** : scikit-learn (modèles simples), CSV de feedbacks
- **Mobile** : Flutter (Dart)
- **Base de données** : PostgreSQL ou Supabase
